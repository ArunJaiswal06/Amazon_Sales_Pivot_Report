# Amazon Sales Dashboard (Excel & Google Sheets)

## Project Overview
This project analyzes Amazon product sales data using Pivot Tables, charts, and interactive dashboards to identify sales trends, customer satisfaction, and profit margin performance across product categories.

## Tools Used
- Microsoft Excel / Google Sheets
- Pivot Tables
- Charts & Slicers

## Key Analysis
- Sales Distribution by Category
- Average Rating by Category
- Profit Margin Analysis

## Insights
- Electronics and Computers categories generate the highest sales and strong profit margins.
- Home & Kitchen shows high customer satisfaction.
- Office Products and Toys & Games have lower margins, suggesting pricing optimization opportunities.
